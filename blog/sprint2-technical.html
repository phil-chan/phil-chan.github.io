What are the best practices associated with using classes vs. ids?

IDs

Best practices: IDs should be unique, there will only ever be one of this in the entire web page 
                       
Example: the main logo of a website, which should be unique will be assigned an id. See below
An id attribute assigned in HTML: id = "mainLogo"
An id attribute assigned the font color red in a seperate CSS file: #mainLogo: {color: red};

Classes

Best practices: Classes can be common, there should be groups of similar elements under each class

Example: there are many blog posts, so they should be assigned a class. See below
A class attribute assigned in HTML: class = "blogPosts"
A class attribute assigned the font color pink in a seperate CSSfile: .blogPosts {color: pink};
